# How to run 
## clone the repository
```bash
git clone https://github.com/nongdancryptos/Functor-Bot.git
cd functot
```
## install the requirements
```bash
npm install
```
## fill your email and password in `account.txt`
```bash
email,pass
```
## fill your proxies in `proxy.txt`
```bash
http://user:pass@ip:port
```
## run the script
```bash
npm run start
```
